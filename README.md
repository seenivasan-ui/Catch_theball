# Catch_theball
A simple Catch the Ball game built using Python and Pygame. Move the basket left and right to catch the falling ball and increase your score.
📌 Features

🏀 Falling ball animation

🧺 Movable basket using keyboard

📈 Score tracking system

⚡ Smooth gameplay with 60 FPS

🎮 Simple and beginner-friendly Python game

🛠️ Technologies Used

Python

Pygame

🎮 How to Play

Run the game.

Use Left Arrow (←) to move basket left.

Use Right Arrow (→) to move basket right.

Catch the falling ball to increase your score.

If the ball falls to the ground → Game Over.

▶️ Installation

Install pygame first:

pip install pygame

Run the game:

python game.py
